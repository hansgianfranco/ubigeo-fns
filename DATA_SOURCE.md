# Data Source

This project uses official geographic data from Peru.

---

## Primary source

All UBIGEO information is based on:

### Instituto Nacional de Estadística e Informática (INEI)

- Country: Peru
- Institution: INEI (Instituto Nacional de Estadística e Informática)
- Data: UBIGEO codes (Departments, Provinces, Districts)

---

## What data is included

- Department codes and names
- Province hierarchy
- District-level UBIGEO codes
- Standardized geographic structure used in Peru

---

## Data usage

This dataset is used to:

- validate UBIGEO codes
- resolve geographic names
- build hierarchical location trees
- support search and filtering functions

---

## Disclaimer

This project does not modify the official structure of UBIGEO data.

All mappings are based on publicly available INEI datasets and are used for developer convenience.

---

## Purpose

The goal of this project is to provide:

- easy-to-use utilities for developers
- accurate geographic validation tools
- fast lookup performance for production systems
